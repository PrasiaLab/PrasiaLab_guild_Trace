# PrasiaLab_guild_Trace

프라시아랩 **추적 연구소** 1차 구성입니다.

## 반영 내용

- 좌측 메뉴는 유지하되 불필요한 안내 항목 제거
- 좌측 하단 캐릭터 이미지 제거
- 메인 화면의 긴 분석 안내 제거
- 화면 내 내부 파일명 노출 최소화
- 비교 데이터 선택 기능 추가
- 결사 유사 후보를 상위권 기준으로 정렬하도록 처리
- 상세 보기 팝업 폭 확장 및 가독성 개선
- 상세 하단에 이전/이후 멤버의 닉네임, 레벨, 직업군 표 추가
- 검색 기능 유지

## 운영 메모

실제 운영 시 비교 결과 데이터와 상위권 정렬 기준 데이터를 갱신하면 화면에 반영됩니다.
